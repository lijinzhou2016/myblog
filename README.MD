# django simple demo


### 环境
* Python 2.7
* Django==1.10


### 本地快速启动

* 1 下载源码到本地
* 2 进入项目目录
* 3 执行： python manage.py runserver
* 4 访问URL：
    首页：127.0.0.1:/8000/blog/list
    后台管理系统 127.0.0.1/8000/admin
* 5 创建超级用户用来登录后台系统：python manage.py createsuperuser
